Sum (I)


This time we discuss the problem that how \[\sum_{k=1}^n{k^\alpha}(\alpha\in N^{*})\] is reduced to a simple \(\alpha+1\) degree polynomial.

First, when \(\alpha=1\) aka 'Gauss Summation Method', pairing and adding.

However, there is another way. As \[\sum_{k=1}^n{[k^2-(k-1)^2]}=n^2\]and\[k^2-(k-1)^2=2k-1\]then\[\sum_{k=1}^n{[k^2-(k-1)^2]}=n^2=\sum_{k=1}^n{(2k-1)}=2\sum_{k=1}^n{k}-n\] so that \[\sum_{k=1}^n{k}=\frac{n(n+1)}{2}\]

From the discussion above, we are able to get a generic solution: adding up \(k^\alpha-(k-1)^\alpha\).

For example, when \(\alpha=3\)\[k^4 - (k-1)^4=4k^3-6k^2+4k-1\]\[n^4=3\sum_{k=1}^n{k^3}-6\sum_{k=1}^n{k^2}+4\sum_{k=1}^n{k}-n\]We already knows the result of \(\displaystyle \sum_{k=1}^n{k}\) and \(\displaystyle \sum_{k=1}^n{k^2}\), so that\[\sum_{k=1}^n{k^3}=\frac{n^2(n+1)^2}{4}\]
